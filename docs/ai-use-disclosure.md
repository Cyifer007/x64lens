# AI Use Disclosure

This project may use AI assistance for planning, documentation review, scaffolding suggestions, code review, test design, and research organization.

Project policy:

- Core assembly implementation should be understood, reviewed, and manually validated by the maintainer.
- AI-generated code must not be accepted blindly.
- Any AI-assisted implementation must be debugged, tested, and explainable by the maintainer.
- AI assistance should support learning, not replace low-level reasoning.
- Public research claims must be based on measured results, not AI-generated assumptions.

Initial scaffold note:

- The initial repository skeleton, documentation contracts, and starter NASM build structure were drafted with AI assistance and should be reviewed, edited, and validated before submission or publication.
