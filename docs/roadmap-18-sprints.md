# Superseded Eighteen-Sprint Roadmap

## Status

Superseded by the canonical
[`roadmap-22-sprints.md`](roadmap-22-sprints.md) in Sprint 10 Patch 053.

## Historical purpose

The eighteen-sprint plan separated parser correctness, mitigation depth,
provenance, primitive expansion, corpus work, benchmarking, triage, automation,
case study, replication, and release. The Patch 053 architecture review retained
those work classes but inserted four explicit stages before the first research
release:

1. diagnostic benchmark foundation;
2. loader and mitigation precision;
3. semantic capability completion;
4. optional decoder and concurrency ablations.

The expansion prevents the corpus and publication campaign from freezing before
material capability and task-definition decisions are made.

Historical documents may continue to reference this file when describing the
roadmap that governed an earlier patch. New planning changes must update
[`roadmap-22-sprints.md`](roadmap-22-sprints.md).
