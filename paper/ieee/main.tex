\begin{abstract}
This paper presents x64lens, an assembly-first ELF64 x86\_64 binary analysis tool designed to identify exploit-relevant code primitives, classify their semantic usefulness, evaluate mitigation context, and produce reproducible reports for offensive research, defensive triage, and binary hardening assessment. The initial study evaluates whether an assembly-first gadget discovery engine can improve runtime and memory efficiency compared to existing tooling while preserving comparable gadget coverage and adding semantic primitive analysis.
\end{abstract}
