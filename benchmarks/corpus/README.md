# Benchmark Corpus

Do not commit proprietary binaries, malware samples, or large generated corpora here.

Recommended corpus tiers:

1. Controlled toy binaries built from source.
2. Linux system binaries available on the benchmark host.
3. Redistributable open-source binaries.
4. Compiler and hardening matrix binaries generated from source.

Every corpus entry used in publication results must be listed in a corpus manifest.
